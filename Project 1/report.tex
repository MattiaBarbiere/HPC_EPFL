\section{Conclusion and further considerations}
In this report I focus on the parallelization of conjugate gradient method for solving linear systems. While the main focus of this project was on parallelizing the \verb|mat_vec| function there are further improvements that one might apply to squeeze out even more performance. The main improvement that comes to mind was the loading of the matrix onto all processors. This is indeed very wasteful when the matrix is large since every processor only uses a faction of the stored data. A remedy to this would be scattering the matrix with \verb|MPI_Scatter| from processor 0 to all the other processors so that each processor would have to store only the part of the matrix it actually uses. Moreover, one would go one step further and directly load the relevant part of the matrix to each processor, avoiding \verb|MPI_Scatter| entirely. These improvements would definitely help with memory efficiency, however the implementation of these approach were outside the scope of this exercise.
